# Guess The Number Game

## 📌 Objective

The objective of this project is to develop a Java console-based game where the computer generates a random number and the player has to guess it within a limited number of attempts. The program provides hints after every guess and calculates the player's score based on performance.

---

## ✨ Features

* Random Number Generation (1–100)
* User-Friendly Console Interface
* High/Low Hints
* Limited Number of Attempts
* Score Calculation
* Multiple Rounds (Play Again Option)
* Input Validation using Scanner

---

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class
* Random Class
* Loops
* Conditional Statements

## ▶️ How to Run

1. Open the project in your Java IDE or Command Prompt.
2. Navigate to the project folder.
3. Compile the program:

```bash
javac GuessTheNumber.java
```

4. Run the program:

```bash
java GuessTheNumber
```

---

## 🎮 Gameplay

1. The computer generates a random number between **1 and 100**.
2. The player enters a guess.
3. The program provides hints:

   * Too High
   * Too Low
4. The player continues guessing until:

   * The correct number is guessed, or
   * The maximum number of attempts is reached.
5. The final score is displayed based on the number of attempts used.
